# LoL Inhouse Companion — Downloads

Public release host for the **Companion** desktop app used with the LoL inhouse Discord bot.
The bot's source lives in a private repo; this repo exists only to make the installer
downloadable anonymously (the Discord `/app` command links to releases here).

## Get the app

- **Latest (always):** [Companion.exe](https://github.com/kevinpsj0/lol-companion-releases/releases/latest/download/Companion.exe)
- All versions: see the [Releases](https://github.com/kevinpsj0/lol-companion-releases/releases) tab.

Windows 10/11, portable single-file `.exe` (no install). On first run Windows SmartScreen
may say "Windows protected your PC" — click **More info → Run anyway**.
